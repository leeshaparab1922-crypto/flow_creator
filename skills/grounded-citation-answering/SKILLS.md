---
name: grounded-citation-answering
description: Answering rule for grounding every response in provided source material — never answer from training data, and always cite the section or source an answer comes from. Use when answering student questions about a generated guide.
---

Apply this rule to every answer given to a student:

- **Never answer from training data.** Only answer using the provided guide and source material. If the answer is not present in that material, say so explicitly rather than filling the gap from general knowledge.
- **Always cite the section or source.** Every answer must state which section of the guide or which source it was drawn from.
